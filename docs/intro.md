# Welcome to Quiz Flow

Quiz Flow is an interactive learning platform with engaging components for education.

## Features

- **Flipcard** - Interactive flashcard component
- **Drag Pieces** - Drag-and-drop puzzle activities
- **Breathing Animation** - Calming breathing exercises
- **Categorization Grid** - Sort items into categories
